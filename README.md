# 🌌 Super Workspace (超级扩展坞) - 新手极客指南

欢迎来到 **Super Workspace**！这不仅是一个投屏软件，而是一个能将你的手机、平板与 Windows 电脑在底层物理级打通的“赛博空间局域网控制中枢”。

无论你是想把平板变成 **120Hz 的电脑副屏**、**无延迟的压感手绘板**、**Stream Deck 宏按键台**，还是想把手机变成 **电脑的高清摄像头和麦克风**，这里都能一键实现。

在开启魔法之前，我们需要花 1 分钟时间，拿到设备的“底层通行证”。

---

## 🗝️ 第一步：唤醒设备的“开发者灵魂”

为了让电脑能通过 ADB (Android Debug Bridge) 协议接管你的设备，你需要在手机/平板上开启**「USB 调试」**。

1. **找到版本号**：打开手机/平板的 `设置` -> `关于手机/关于平板`。
2. **疯狂点击**：连续快速点击 `版本号`（MIUI/HyperOS 是点击“OS版本”，鸿蒙是点击“版本号”）**7 次**，直到屏幕下方提示“您现在已处于开发者模式！”。
3. **开启调试**：退回到设置首页，找到新出现的 `系统和更新` -> `开发者选项`（有些系统在“更多设置”里）。
4. **拨下开关**：在开发者选项中，找到并开启 **`USB 调试`**（如果你的手机有“仅充电模式下允许 ADB 调试”或“USB调试(安全设置)”，请一并打开）。

---

## 🔌 第二步：有线破冰 (首次连接)

1. 使用数据线，将你的手机/平板连接到电脑。
2. **极其重要**：此时你的手机/平板屏幕上会弹出一个提示框：**「是否允许 USB 调试？」**。
3. 请务必勾选 **“始终允许使用这台计算机进行调试”**，然后点击 **「允许/确定」**。
   *(如果你错过了这个弹窗，软件右下角会提示“等待授权 (unauthorized)”，请拔插数据线重新触发弹窗！)*
4. 打开 `Super Workspace` 电脑端软件，观察右侧的【设备管理中枢】。如果出现了 `🔌 USB 设备 [你的设备型号]`，恭喜你，物理桥梁已搭通！

---

## 🪄 第三步：剪断线缆 (一键无线模式)

谁想一直拖着一根数据线呢？让我们把它变成无线的！

1. 确保你的手机/平板 和 电脑连接在 **同一个 Wi-Fi 路由器下**。
2. 在软件右侧的【设备管理中枢】中选中你的设备，然后点击紫色的魔法按钮：
   👉 **【🪄 一键切换为无线模式】**
3. 稍等 2 秒钟，如果右下角弹出绿色提示“✅ 无线通道已成功建立！”。
4. **拔掉数据线吧！** 此时你的设备型号前会变成 `📶 无线设备`。你可以拿着平板去任何地方，所有的功能依然满血运行！

---

## 🚀 第四步：探索九大核心魔法

当设备连接成功后，你可以点击左侧的九大功能卡片，探索不同的生产力形态：

### 🖥️ 视觉系拓展
- **📱 手机镜像 (PC Receiver)**：以最高 120帧、极低延迟将手机画面投射到电脑上，支持电脑键鼠直接控制手机。
- **🖥️ 扩展电脑副屏 (Display Ext.)**：在 Windows 内存中捏造一块虚拟显示器，并把画面投给平板。你的平板瞬间变成一块支持触控的电脑第二显示器！
- **🪟 单窗口镜像 (App Mirror)**：只想把老板看不到的游戏窗口投到平板上？用这个！

### 🪄 物理外设幻化
- **🪄 触控板模式 (Magic Trackpad)**：把平板变成苹果妙控板，支持单指移动、双指滚动、三指切屏、四指切桌面等极其丝滑的手势！
- **🎛️ Stream Deck (自定义控制台)**：把平板变成你的专属快捷键键盘。一键运行代码、一键静音、带有炫酷的物理旋钮甚至原神自动清日常宏！
- **🎨 数位板映射 (Graphics Tablet)**：画师福音！把平板变成 Wacom 压感板，支持 2048 级真实物理压感，在 Photoshop 里画出完美的笔锋。

### 🎧 音视频路由
- **🎥 虚拟摄像头 (Virtual Camera)**：把手机那颗一亿像素的后置主摄，变成电脑的摄像头。去惊艳腾讯会议里的同事吧！
- **🎤 全景音频路由 (Audio Routing)**：赛博时代的终极调音台！把手机的音乐零延迟投给电脑音响，或者把电脑的声音发给平板，带动态电平表和专业 EQ 调节。

## ❓ 常见问题排雷 (FAQ)

**Q1: 为什么点击“开启数位板/控制台”后，平板上没有画面，显示找不到网页？**
- A: 这通常是因为安卓系统的后台限制。请手动打开平板上的浏览器（推荐 Edge/Chrome），并在地址栏输入电脑弹窗提示的 `http://localhost:端口号`。

**Q2: 为什么开启“虚拟副屏”后，平板上显示的画面和电脑主屏一模一样（镜像），而不是扩展屏幕？**
- A: 虚拟屏幕创建成功后，请在电脑键盘上按下 `Win + P`，在右侧弹出的侧边栏中选择 **“扩展”** 即可！

**Q3: 为什么音频路由里有刺耳的回声？**
- A: 请不要将“电脑麦克风”直接连向“电脑扬声器”，这会产生物理级的声学啸叫。如果遇到死循环，软件会触发紫色的警告弹窗自动阻断。

**Q4: 为什么插上数据线了，设备上却死活不弹出“允许 USB 调试”的窗口？**
- A: 这是 Android 极客圈最常见的“坑”：你的电脑缺少该品牌的 **ADB 专属底层驱动**，导致电脑和设备无法通过 ADB 协议“握手”。
  **终极解法**：去该品牌的官网，下载并安装一次官方电脑端软件（如：vivo 办公套件、华为电脑管家、小米助手）。安装时，官方套件会自动把底层驱动注入 Windows 系统。只要安装完成，弹窗就会立刻出现。之后你就算把官方套件卸载掉，驱动也会保留，可以永久顺畅使用 Super Workspace！
